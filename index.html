指南 #

介绍 #

基于 React 17 移动端组件库。

特性 #

  • 提供 70+ 套组件，覆盖移动端各类场景
  • 支持主题定制
  • 支持黑暗模式
  • 支持国际化
  • 支持 TypeScript

支持环境 #

  • Android 5.0+
  • iOS 10.0+
  • Node.js 12.0+

框架版本 #

  • react >= 16.9.0 & < 18.0.0
  • react-dom >= 16.9.0 & < 18.0.0

安装 #

# npm
npm i -S arkui-mobile-react

# yarn
yarn add arkui-mobile-react

# pnpm
pnpm add arkui-mobile-react
